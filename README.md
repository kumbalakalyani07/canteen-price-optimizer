# Campus Canteen Price Optimizer

This project finds the best price for a product to maximize profit 
using demand and cost.

## Features
- GUI using Tkinter
- Calculates demand equation
- Shows revenue & profit table
- Suggests best price

## How to run
1. Install Python
2. Run:
   python price_optimizer.py
